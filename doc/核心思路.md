### 第一部分：模型背景与核心变量设定

在建模中，我们需要把心理学现象转化为数学语言。心理学中著名的**“时间动机理论”（Temporal Motivation Theory, TMT）**指出：人对完成某件事的动机，与任务的价值成正比，与完成任务所需的时间（延迟）成反比。

基于此，我们定义以下变量：
*   $t$：当前时间（天）。
*   $T$：任务的截止期限（Deadline），即在 $t=T$ 时必须交作业。
*   $x(t)$：在 $t$ 时刻已经完成的作业比例，取值范围 $x \in [0, 1]$（0代表完全没动笔，1代表做完了）。
*   $A(t)$：学生在 $t$ 时刻感受到的“焦虑/压力值”，压力越大，行动力越强。

---

### 第二部分：基础模型（单变量常微分方程）—— 纯解析解

为了体现你的数学推导能力，我们可以先建立一个相对简单、但能求出**绝对精确解析解**的模型。

**1. 假设与方程构建**
我们假设学生做作业的速率 $\frac{dx}{dt}$（即每天做多少作业）取决于两个因素：
1.  **剩余的作业量** $(1 - x)$：剩下的越多，做的空间越大；如果做完了（$x=1$），速率自动变为0。
2.  **时间的紧迫感**：距离Deadline越近（$T-t$ 越小），紧迫感越强。我们引入“拖延系数” $k$。

构建微分方程：
$$ \frac{dx}{dt} = c \cdot (1 - x) \cdot \frac{1}{1 + k(T - t)} $$
*参数解释：*
*   $c$：学生的基础执行力常数。
*   $k$：拖延系数（延迟折扣率）。$k$ 越大代表这个人越爱拖延（因为当 $T-t$ 较大时，分母很大，导致做作业速率极小；只有当 $T-t$ 接近0时，速率才会飙升）。

**2. 详细的方程求解（可直接写进论文的推导过程）**
这是一个典型的一阶可分离变量微分方程，非常适合展示高中微积分能力。
我们将含有 $x$ 的项放在左边，含有 $t$ 的项放在右边：
$$ \frac{dx}{1 - x} = \frac{c}{1 + k(T - t)} dt $$

两边同时积分：
$$ \int \frac{1}{1 - x} dx = \int \frac{c}{1 + k(T - t)} dt $$

左边积分（设 $u = 1-x, du = -dx$）：
$$ -\ln|1 - x| = -\frac{c}{k} \ln|1 + k(T - t)| + C $$

两边去掉负号并取指数：
$$ 1 - x = e^{-C} \cdot \left( 1 + k(T - t) \right)^{\frac{c}{k}} $$

设 $C_1 = e^{-C}$。利用初始条件：在 $t=0$ 时，作业完成度 $x=0$。代入得：
$$ 1 - 0 = C_1 \cdot (1 + kT)^{\frac{c}{k}} \implies C_1 = \frac{1}{(1 + kT)^{\frac{c}{k}}} $$

将 $C_1$ 代回原式，整理得到**最终的模型解析解**：
$$ x(t) = 1 - \left( \frac{1 + k(T - t)}{1 + kT} \right)^{\frac{c}{k}} $$

**3. 模型结论与亮点**
这个公式非常漂亮！你可以向评委展示：当 $t \to T$ 时（到了Deadline当天），完成度 $x(T) = 1 - \left(\frac{1}{1+kT}\right)^{\frac{c}{k}}$。
*注意：这说明如果执行力 $c$ 太低，或者拖延系数 $k$ 太高，学生在Deadline当天**有可能写不完作业！**（$x(T) < 1$），这完美符合现实！*

### 第三部分：进阶模型（“焦虑-行动”二维动力系统）—— 计算机仿真

为了拿高分，你可以说：“基础模型虽然有解析解，但人是情绪动物。因此我们引入**焦虑值 $A(t)$**，构建一个包含情绪反馈的二维动力系统。”

**1. 方程组构建**
*   **行动方程**：做作业的速度，正比于当前的焦虑程度，并且受限于剩余作业量。
    $$ \frac{dx}{dt} = \mu \cdot A(t) \cdot (1 - x(t)) $$
    *（$\mu$ 是行动转化率：焦虑转化为行动的能力）*

*   **焦虑方程**：焦虑值的变化率 = 焦虑的产生 - 焦虑的自我缓解。
    $$ \frac{dA}{dt} = \frac{\alpha \cdot (1 - x(t))}{1 + \gamma(T - t)} - \beta \cdot A(t) $$
    *（$\alpha$ 是对任务敏感度；$\gamma$ 是拖延患者的“时间麻木系数”；$\beta$ 是自我安慰、打游戏逃避导致焦虑下降的速率）*

**2. 为什么这个进阶模型好？**
这是一个非线性的常微分方程组（ODEs），没有简单的解析解，必须用计算机（Python）进行数值求解。这能极大提升你论文的技术含量。
*   对于**严重拖延症患者**（$\gamma$ 很大）：早期时间 $T-t$ 很大，第一项（产生焦虑）很小，焦虑值 $A(t)$ 维持在接近0的水平，学生愉快地玩耍。当 $t$ 逼近 $T$ 时，分母迅速变小，焦虑值暴增，带动 $\frac{dx}{dt}$ 飙升，产生经典的**“死线前疯狂补作业”（曲棍球棒曲线）**现象。

### 第四部分：Python 仿真代码具体怎么写

你可以把下面这段代码放在论文附录，它会生成非常漂亮的两张图（作业进度曲线 和 焦虑值变化曲线），对比“非拖延者”和“严重拖延者”。

请确保你安装了 `numpy`, `scipy`, 和 `matplotlib`（命令行运行 `pip install numpy scipy matplotlib`）。

```python
import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

# ----------------- 1. 定义二维动力系统 -----------------
def procrastination_model(y, t, mu, alpha, gamma, beta, T):
    """
    y[0]: x(t) 任务完成度 (0 到 1)
    y[1]: A(t) 焦虑值/压力值
    """
    x, A = y
    
    # 限制 x 最大为 1 (作业写完了就不写了)
    if x >= 1.0:
        dxdt = 0.0
        # 作业写完了，焦虑随着时间自然消退
        dAdt = -beta * A 
    else:
        # 1. 行动方程：进度增加速率 = 转化率 * 焦虑值 * 剩余任务量
        dxdt = mu * A * (1 - x)
        
        # 2. 焦虑方程：焦虑增加 = 剩余任务带来的时间压力 - 自我情绪缓解
        # 使用 1 + gamma*(T-t) 来体现“时间折扣”（拖延症本质）
        time_pressure = (alpha * (1 - x)) / (1 + gamma * (T - t))
        dAdt = time_pressure - beta * A
        
    return [dxdt, dAdt]

# ----------------- 2. 参数设置 -----------------
T_deadline = 14.0       # 假设作业期限为14天
t_steps = np.linspace(0, T_deadline, 500) # 生成500个时间点
y0 = [0.0, 0.0]         # 初始状态：进度为0，焦虑值为0

# 通用参数
mu = 0.4    # 行动力转化率
alpha = 1.0 # 压力敏感度
beta = 0.8  # 焦虑自我缓解率（例如通过玩手机逃避）

# 对比参数：非拖延症 vs 严重拖延症
gamma_diligent = 0.1  # 勤奋学生：几乎不受时间远近影响
gamma_procrast = 4.0  # 拖延学生：距离远时极度麻木，对时间极度打折

# ----------------- 3. 微分方程数值求解 -----------------
sol_diligent = odeint(procrastination_model, y0, t_steps, 
                      args=(mu, alpha, gamma_diligent, beta, T_deadline))
sol_procrast = odeint(procrastination_model, y0, t_steps, 
                      args=(mu, alpha, gamma_procrast, beta, T_deadline))

x_diligent, A_diligent = sol_diligent[:, 0], sol_diligent[:, 1]
x_procrast, A_procrast = sol_procrast[:, 0], sol_procrast[:, 1]

# ----------------- 4. 绘图展示 -----------------
# 开启正常显示中文的设置
plt.rcParams['font.sans-serif'] = ['SimHei'] # Windows用SimHei，Mac可用Arial Unicode MS
plt.rcParams['axes.unicode_minus'] = False 

fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 8))

# 图1：任务完成度对比
ax1.plot(t_steps, x_diligent, 'b-', linewidth=2.5, label='勤奋型 (低拖延系数)')
ax1.plot(t_steps, x_procrast, 'r-', linewidth=2.5, label='拖延型 (高拖延系数)')
ax1.axhline(y=1.0, color='gray', linestyle='--')
ax1.axvline(x=T_deadline, color='k', linestyle=':', label='Deadline')
ax1.set_ylabel('作业完成进度 x(t)', fontsize=12)
ax1.set_title('作业进度动态变化曲线', fontsize=14)
ax1.legend(loc='upper left')
ax1.grid(True, alpha=0.3)

# 图2：焦虑值对比
ax2.plot(t_steps, A_diligent, 'b-', linewidth=2.5, label='勤奋型焦虑值')
ax2.plot(t_steps, A_procrast, 'r-', linewidth=2.5, label='拖延型焦虑值')
ax2.axvline(x=T_deadline, color='k', linestyle=':')
ax2.set_xlabel('时间 t (天)', fontsize=12)
ax2.set_ylabel('心理焦虑值 A(t)', fontsize=12)
ax2.set_title('心理焦虑/压力动态变化曲线', fontsize=14)
ax2.legend(loc='upper left')
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()
```
