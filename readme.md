## 一、建模计算的基本流程

1. **明确模型变量与方程**  
   确定你打算用几个微分方程（或差分方程）描述状态变化。  
2. **选择求解方法**  
   - 如果是常微分方程（ODE），用 `scipy.integrate.solve_ivp` 或自己写欧拉法。  
   - 如果是含切换条件的混合系统，可在每个时间步判断条件，手动推进。  
3. **参数设置**  
   先根据心理假设设定一组“合理”的参数（例如恐慌阈值=0.8，自控力消耗速率=0.1等）。  
4. **运行模拟**  
   得到各状态变量随时间变化的数值解。  
5. **结果可视化与分析**  
   绘制时间序列图、相图、敏感性分析等。

## 三、一个简单的入门模型：单截止期限下的工作进度与焦虑

为了让你快速上手，我们先构建一个**简化版**的动力学模型。它包含两个状态变量：

- $ W(t) $ ：已完成的工作量（归一化，0→1）  
- $ A(t) $ ：焦虑水平（0~1，越高越焦虑）

基本假设：
- 焦虑随剩余时间减少而自然增长，且当工作量落后时焦虑加剧。
- 只有当焦虑超过某个阈值（如0.6）时，学生才开始工作。
- 工作时，工作进度与焦虑水平成正比（焦虑推动行动，但过度焦虑反而可能降低效率，我们暂且简单线性）。
- 工作时焦虑会因进展而缓解。

**模型方程**（分段）：

**当焦虑 < 阈值 $A_{\text{th}}$ （拖延阶段）**：
$$
\frac{dW}{dt} = 0, \quad \frac{dA}{dt} = \frac{\alpha}{D - t} - \beta W
$$
（焦虑随时间增加，同时因已完成工作量$W$的积累而轻微缓解）

**当焦虑 ≥ 阈值 $A_{\text{th}}$ （工作阶段）**：
$$
\frac{dW}{dt} = \gamma A, \quad \frac{dA}{dt} = \frac{\alpha}{D - t} - \beta W - \delta \cdot \frac{dW}{dt}
$$
（焦虑来源仍是时间压力，但工作会进一步降低焦虑，$\delta$ 为工作效率对焦虑的抑制系数）

这里 $D$ 是截止期限（例如设定 $D=10$ 天），$t$ 从0开始。注意当 $t \to D$ 时分母趋近0，需在数值上做截断（在 $t = D-0.001$ 停止）。

## 四、代码实现步骤（使用欧拉法手动推进，方便理解切换逻辑）

我们将采用**离散时间欧拉法**，每个时间步判断当前状态（拖延还是工作），然后更新变量。这样容易加入条件切换。

### 1. 导入库

```python
import numpy as np
import matplotlib.pyplot as plt
```

### 2. 定义参数和初始条件

```python
# 参数
D = 10.0            # 截止期限（天）
alpha = 0.5         # 时间压力系数
beta = 0.05         # 已完成工作对焦虑的缓解系数
gamma = 0.3         # 工作效率系数（工作时的进度速率）
delta = 0.4         # 工作时焦虑抑制系数
A_th = 0.6          # 焦虑阈值（超过此值开始工作）

# 初始条件
W0 = 0.0
A0 = 0.1            # 初始轻微焦虑

# 时间离散
dt = 0.01           # 时间步长（天）
t_max = D - 0.001   # 避免最后时刻除以零
t = np.arange(0, t_max, dt)
n_steps = len(t)

# 存储结果
W = np.zeros(n_steps)
A = np.zeros(n_steps)
W[0] = W0
A[0] = A0
```

### 3. 主循环：根据当前焦虑值选择微分方程

```python
for i in range(1, n_steps):
    t_curr = t[i-1]
    W_curr = W[i-1]
    A_curr = A[i-1]
    
    # 计算当前焦虑源（时间压力项）
    time_pressure = alpha / (D - t_curr)
    
    if A_curr < A_th:
        # 拖延阶段：不工作
        dWdt = 0.0
        dAdt = time_pressure - beta * W_curr
    else:
        # 工作阶段
        dWdt = gamma * A_curr
        dAdt = time_pressure - beta * W_curr - delta * dWdt
        # 确保焦虑不会变成负数（最低为0）
        if dAdt < 0 and A_curr + dAdt * dt < 0:
            dAdt = -A_curr / dt   # 强制降至0
    
    # 欧拉更新
    W[i] = W_curr + dWdt * dt
    A[i] = A_curr + dAdt * dt
    
    # 边界条件：工作进度不能超过1，焦虑保持在[0,1]区间
    W[i] = min(W[i], 1.0)
    A[i] = np.clip(A[i], 0.0, 1.0)
    
    # 如果工作已完成，提前终止
    if W[i] >= 1.0 - 1e-6:
        # 截断后续数据
        W[i:] = 1.0
        A[i:] = A[i]
        break
```

### 4. 绘制结果

```python
plt.figure(figsize=(12, 4))
plt.subplot(1,2,1)
plt.plot(t[:i+1], W[:i+1], label='Work progress')
plt.xlabel('Time (days)')
plt.ylabel('W (completed)')
plt.ylim(0,1)
plt.grid(True)

plt.subplot(1,2,2)
plt.plot(t[:i+1], A[:i+1], label='Anxiety', color='orange')
plt.axhline(y=A_th, color='r', linestyle='--', label='Threshold')
plt.xlabel('Time (days)')
plt.ylabel('Anxiety')
plt.grid(True)
plt.legend()
plt.tight_layout()
plt.show()
```

运行这段代码，你会看到：  
- 前期焦虑缓慢增长，但未超过阈值时进度不变（拖延）。  
- 当焦虑突破阈值后，开始工作，进度上升，焦虑因工作而下降（可能短暂波动）。  
- 最终在截止期前完成全部作业。

## 五、如何进一步扩展

你可以在上述基础上逐步添加复杂性：

1. **加入自控力储备变量 $S(t)$**  
   增加一个方程，工作时消耗自控力，自控力低时效率下降（如将 $\gamma$ 改为与 $S$ 成正比）。  
2. **状态切换时的突变**  
   当焦虑达到阈值时，可以瞬间改变参数（如增加初始工作效率）。  
3. **随机扰动**  
   在每个时间步给 $A$ 添加一个小的随机噪声，模拟情绪波动。  
4. **多截止期对比**  
   将 $D$ 改为多个中间节点，模拟阶段性作业。  
5. **参数敏感性分析**  
   用循环遍历不同参数（如 $\alpha, \gamma$），观察完成时间、最大焦虑等指标的变化。

## 六、关键代码解释

- **欧拉法**：$ y_{n+1} = y_n + f(y_n, t_n) \cdot \Delta t $  
  对新手最直观，但精度有限。若需更精确，可使用 `scipy.integrate.solve_ivp`，但处理分段条件稍麻烦。  
- **条件判断**：在每个时间步根据当前 $A$ 值决定使用哪组方程，这模拟了“从拖延到工作的切换”。  
- **边界处理**：确保 $W$ 不超过1，$A$ 非负，避免数值发散。

---

## 七、常见问题与调试

- **参数选择**：如果 $A$ 始终达不到阈值，说明 $\alpha$ 太小或 $\beta$ 太大，可适当调整。  
- **数值不稳定**：时间步长 $dt$ 不要太大（0.01 通常安全）。如果出现负值，加入 clip 操作。  
- **截止期前发散**：在 $t$ 接近 $D$ 时，压力项 $ \alpha/(D-t) $ 会趋于无穷，所以我们将 $t_{\max}$ 设为略小于 $D$，并确保工作能在之前完成。
